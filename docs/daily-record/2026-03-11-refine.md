# 2026-03-11 추가 보완 (refine)

## 개요

강사님 구현 코드 리뷰를 통해 현재 구조와의 차이를 파악하고,
`agent_service.py`와 `stock_agent.py`를 강사님과 동일한 구조로 리팩토링.

---

## 변경 전/후 구조 비교

| 항목 | 변경 전 | 변경 후 |
|---|---|---|
| LLM 초기화 위치 | `stock_agent.py`의 `Agent.__init__` | `AgentService.__init__` |
| Checkpointer 초기화 | 모듈 레벨 `_checkpointer = MemorySaver()` | `async _init_checkpointer()` 메서드 (첫 호출 시 1회) |
| 에이전트 생성 방식 | `Agent` 클래스 + 직접 LangGraph StateGraph 구성 | `create_stock_agent()` 팩토리 함수 + `create_agent()` 활용 |
| ChatResponse 정의 | `@tool` 데코레이터 | `@dataclass` |
| 응답 포맷 처리 | 수동 노드(`execute_chat_response`) + fallback 래핑 | `ToolStrategy(ChatResponse)` |
| stream_mode | `astream()` 내부 고정 | `agent_service.py` 호출부에서 명시 |

## 수정 내용

### `app/agents/stock_agent.py` — 전면 재작성

**변경 전**: `Agent` 클래스 내부에서 LangGraph `StateGraph`를 직접 구성

**변경 후**: `create_stock_agent()` 팩토리 함수로 전환, `langchain.agents.create_agent()` 활용

```python
# ChatResponse: @tool → @dataclass
@dataclass
class ChatResponse:
    message_id: str
    content: str
    metadata: dict[str, Any]

# 팩토리 함수
def create_stock_agent(
    model: ChatOpenAI,
    checkpointer: BaseCheckpointSaver[Any] = None,
):
    if checkpointer is None:
        checkpointer = MemorySaver()

    return create_agent(
        model=model,
        tools=[get_stock_price, get_company_info, get_recent_news],
        system_prompt=system_prompt,
        response_format=ToolStrategy(ChatResponse),
        checkpointer=checkpointer,
    )
```

제거된 코드:
- `Agent` 클래스 전체
- 수동 구성한 `StateGraph`, `model` 노드, `tools` 노드, `execute_chat_response` 노드
- `should_continue` 라우팅 함수
- `model` 노드 내 ChatResponse fallback 래핑 로직
- 모듈 레벨 `_checkpointer = MemorySaver()` 싱글턴

### `app/services/agent_service.py`

**`__init__`**: LLM 초기화 추가

```python
def __init__(self):
    self.model = ChatOpenAI(
        model=settings.OPENAI_MODEL,
        api_key=SecretStr(settings.OPENAI_API_KEY),
    )
    self.checkpointer = None
    self.agent = None
    self.progress_queue = asyncio.Queue()
```

**`_init_checkpointer()` 추가**: checkpointer 비동기 초기화 (첫 호출 시 1회)

```python
async def _init_checkpointer(self):
    if self.checkpointer is not None:
        return
    from langgraph.checkpoint.memory import MemorySaver
    self.checkpointer = MemorySaver()
```

**`_create_agent()` 수정**: `create_stock_agent()` 팩토리 함수 호출로 변경

```python
def _create_agent(self, thread_id=None):
    assert self.checkpointer is not None, "..."
    if self.agent is not None:
        return
    from app.agents.stock_agent import create_stock_agent
    self.agent = create_stock_agent(
        model=self.model,
        checkpointer=self.checkpointer,
    )
```

**`process_query()`**: `_init_checkpointer()` 호출 추가 및 `stream_mode="updates"` 복원

```python
await self._init_checkpointer()
self._create_agent(thread_id=thread_id)

agent_stream = self.agent.astream(
    {"messages": [HumanMessage(content=user_messages)]},
    config={"configurable": {"thread_id": str(thread_id)}},
    stream_mode="updates",
)
```
